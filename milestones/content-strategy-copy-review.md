# CoreCraft - Content Strategy and Copy Review

This document reviews the current website copy page by page and proposes clearer, more professional content without changing the offer or making unsupported claims.

## Site Name: CoreCraft

### 1. Page Purpose

Define the teaching company as a fundamentals-first programming education brand.

### 2. Audience

Students, early-career developers, and self-taught learners who want structured guidance in programming fundamentals.

### 3. Current Issues

- The name is strong, but the site should explain it more consistently.
- "Core" and "Craft" imply fundamentals and skill, but this meaning is not always made explicit.
- Some copy says "build from the core" repeatedly, which can become repetitive.

### 4. Content Audit

CoreCraft works because it signals depth, discipline, and practical skill. The brand should consistently connect the name to the promise: learn the core ideas behind programming, then apply them through guided practice.

### 5. Recommended Structure

- Brand line: "Programming fundamentals, taught with practical guidance."
- Supporting line: "CoreCraft helps students understand how code works before they move into frameworks, stacks, and advanced tracks."
- Use "fundamentals", "guided practice", "clear reasoning", and "structured path" as repeated terminology.

### 6. Complete Rewritten Copy

**CoreCraft**

Programming fundamentals, taught with practical guidance.

CoreCraft helps students build a reliable foundation in programming through private mentorship, private courses, or group learning. The focus is simple: understand how code works, practice with feedback, and build the confidence to learn any next technology with more clarity.

### 7. Recommended CTAs

- Book a free intro call
- Explore courses
- Find your starting point

### 8. SEO Suggestions

- Homepage title: `CoreCraft | Programming Fundamentals and Mentorship`
- Use "programming fundamentals", "coding mentorship", and "private programming courses" naturally.

### 9. UX Content Improvements

- Add a short brand explanation near the homepage hero or About page.
- Avoid overusing "core" in every section heading.

### 10. Additional Recommendations

Keep the name. It fits the offer and can support a premium education brand if the surrounding copy stays specific and disciplined.
