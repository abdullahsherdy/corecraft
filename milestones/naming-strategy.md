# Company Naming Strategy

Working document for choosing a replacement name for CoreCraft.

## Business Definition

The company is a coding education service business, not only a course catalog.

Services:

- Private mentorship
- Private courses
- Group courses

Teaching focus:

- Intro to Programming
- Object-Oriented Programming
- Functional Programming
- Data Structures and Algorithms

Positioning idea:

- Fundamentals first
- Understand why code works
- Learn with guidance and practice
- Build the base before moving into frameworks, stacks, or advanced tracks

## Why Replace CoreCraft

CoreCraft has a good internal meaning: core fundamentals plus craft/skill. The issue is external discoverability and category confusion.

Problems:

- "Craft" strongly overlaps with Minecraft and gaming-server naming patterns.
- The name can sound like a game server, building game, or craft brand before it sounds like coding education.
- Search results for "CoreCraft" are not clean enough for a new education brand.
- The name does not immediately communicate mentorship, learning, programming, or structured guidance.

## Naming Criteria

A strong name should be:

- Easy to say in Arabic and English.
- Easy to spell after hearing it once.
- Distinct enough to search on Google.
- Connected to programming, logic, foundations, or guided learning.
- Broad enough for mentorship, private courses, and group courses.
- Not limited to one course, one stack, or one age group.
- Professional enough for a premium education service.

Avoid:

- Names that sound like Minecraft servers or game communities.
- Names that are too generic, such as Code Academy, Dev School, or Programming Hub.
- Names tied to one technology, such as DotNet, JavaScript, or Python.
- Names that overpromise outcomes.

## Strong Naming Territories

### 1. Root / Base

Best for the fundamentals-first idea.

Examples:

- BaseLoop
- RootCode
- CodeRoot
- LogicRoot
- StackRoot

### 2. Logic / Reasoning

Best for clear thinking, problem solving, and understanding why code works.

Examples:

- LogicRoot
- LogicPath
- CodeLogic
- ClearLogic

### 3. Guided Learning

Best for mentorship and service-led education.

Examples:

- CodePathway
- MentorStack
- GuidedCode
- CodeGuide

### 4. Practice / Skill

Best for hands-on learning without sounding course-only.

Examples:

- BaseLoop
- CodeDrill
- SkillRoot
- PracticeCode

## Initial Shortlist

### BaseLoop

Meaning: Learn the base, then practice through repeated loops.

Strengths:

- Short and memorable.
- Programming-adjacent because "loop" is a basic programming concept.
- Broad enough for mentorship, private courses, and groups.
- Less likely to sound like a Minecraft server.

Risks:

- Slightly abstract; needs a clear tagline.

Suggested tagline:

Programming fundamentals with guided practice.

### LogicRoot

Meaning: Build programming from the root logic.

Strengths:

- Strong fit for fundamentals and reasoning.
- Professional and clear.
- Works well for OOP, FP, DSA, and problem solving.

Risks:

- More serious, less catchy than BaseLoop.

Suggested tagline:

Understand the logic behind code.

### CodeRoot

Meaning: The root of coding knowledge.

Strengths:

- Very clear.
- Easy to understand.
- Strong connection to fundamentals.

Risks:

- More generic than BaseLoop or LogicRoot.
- May be harder to own in search long term.

Suggested tagline:

Start from the root of programming.

### RootCode

Meaning: Programming from the root up.

Strengths:

- Distinct and direct.
- Good fit for a fundamentals-led brand.

Risks:

- Slightly less natural than CodeRoot.

Suggested tagline:

Build programming skills from the root.

### FundaCode

Meaning: Fundamentals plus code.

Strengths:

- Distinct and searchable.
- Communicates foundations directly.
- Friendly and easy to remember.

Risks:

- Slightly informal.
- "Funda" may feel less premium depending on audience.

Suggested tagline:

Programming fundamentals made clear.

## Current Best Directions

Recommended top three for further validation:

1. BaseLoop
2. LogicRoot
3. CodeRoot

Best strategic fit:

- BaseLoop if we want a catchy, product-like name.
- LogicRoot if we want a premium, serious education brand.
- CodeRoot if we want the clearest fundamentals signal.

## Next Validation Steps

- Search exact name and common variants.
- Check domain availability.
- Check social handle availability.
- Check trademark risk in target markets.
- Test pronunciation with Arabic and English speakers.
- Test whether a new student understands the name without explanation.
